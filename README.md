## 실행 방법
1. 의존성 설치
```
npm install

# 또는

yarn
```

2. 실행 및 브라우저(http://localhost:5173/)에서 확인
```
npm run dev

# 또는

yarn dev
```

## 사용 라이브러리

1. Tailwind CSS

2. TanStack Query(React Query)
    - 서버 API 등, 비동기 데이터를 불러오고 지속적으로 업데이트, 캐싱을 하는데 도움을 주는 라이브러리
    - 비동기 요청 및 상태 업데이트에 대한 보일러플레이트 코드를 줄이고 비동기 요청에 따라오는 요청에 대한 상태를 더 쉽게 관리하기 위해 사용
